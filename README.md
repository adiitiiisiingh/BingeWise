# BingeWise

A movie recommendation web application built using Flask, SQLite, Pandas, Scikit-learn and the OMDb API.

## Features

- Search Movies
- View Movie Details
- Genre-Based Recommendations
- Add Movies to Watchlist
- Remove Movies from Watchlist
- SQLite Database Integration

## Technologies

- Python
- Flask
- SQLite
- HTML/CSS
- Pandas
- Scikit-Learn
- OMDb API
